1. ActiveSQLTransactionError
    Ошибка возникает, когда вы пытаетесь выполнить SQL-запрос,
    но уже существует активная транзакция
2. AdminShutdownError
    Эта ошибка возникает, когда сервер PostgreSQL был остановлен
    администратором
3. AmbiguousAliasError
    Ошибка возникает, когда в SQL-запросе используется неоднозначное
    имя для столбца или таблицы
4. AmbiguousColumnError
    Происходит, когда в запросе есть несколько столбцов с одинаковыми
    именами из разных таблиц
11. CannotCoerceError
    Ошибка возникает, когда PostgreSQL не может преобразовать значение одного
    типа данных в другой тип данных
12. CannotConnectNowError
    Эта ошибка указывает на то, что соединение с базой данных не может быть
    установлено в данный момент
14. CardinalityViolationError
    Происходит при нарушении ограничений кардинальности (например, при попытке
    вставить больше строк в таблицу, чем это разрешено)
16. CharacterNotInRepertoireError
    Эта ошибка указывает на то, что символ не поддерживается кодировкой базы данных
17. CheckViolationError
    Происходит при нарушении ограничения проверки (CHECK) на уровне таблицы
18. ClientCannotConnectError
    Ошибка возникает, когда клиентская программа не может установить соединение с
    сервером базы данных по различным причинам (например, неправильные учетные данные)
19. ClientConfigurationError
    Указывает на ошибку конфигурации клиента базы данных, которая мешает успешному
    подключению или выполнению запросов
20. CollationMismatchError
    Ошибка возникает при несоответствии сортировки между столбцами или выражениями
    в запросе
22. ConfigurationLimitExceededError
    Ошибка указывает на то, что превышены лимиты конфигурации PostgreSQL (например,
    максимальное количество подключений)
23. ConnectionDoesNotExsitError
    Возникает при попытке взаимодействия с соединением, которое было закрыто или
    никогда не существовало
24. ConnectionFailureError
    Эта ошибка указывает на общую проблему подключения к серверу базы данных
25. ConnectionRejectionError
    Ошибка возникает, когда сервер отклоняет запрос на подключение из-за настроек
    безопасности или превышения лимитов
29. DataError
    Общая ошибка для случаев некорректных данных (например, неверный формат даты)
31. DatatypeMismatchError
    Ошибка возникает, когда тип данных, переданный в запросе, не соответствует ожидаемому
    типу данных в базе данных
34. DependentObjectsStillExistError
    Ошибка возникает при попытке удалить объект (например, таблицу или схему), на который
    ссылаются другие объекты (например, внешние ключи)
38. DivisionByZeroError
    Происходит при попытке деления на ноль в SQL-запросе.
42. DuplicateDatabaseError
    Происходит при попытке создать базу данных с именем, которое уже существует
46. DuplicateObjectError
    Общая ошибка для случаев, когда вы пытаетесь создать объект (таблицу, индекс и т.д.) с
    именем, которое уже существует
49. DuplicateTableError
    Происходит при попытке создать таблицу с именем, которое уже существует в базе данных
50. ErrorInAssignmentError
    Ошибка возникает при некорректном присваивании значений в SQL-запросе
84. FeatureNotSupportedError
    Ошибка возникает, когда запрашиваемая функция или возможность не поддерживается текущей
    версией PostgreSQL
86. ForeignKeyViolationError
    Ошибка возникает при нарушении ограничения внешнего ключа (FOREIGN KEY), например, при
    попытке удалить запись, на которую ссылаются другие записи
89. GroupingError
    Происходит при ошибках в группировке данных в запросе
92. IdleSessionTimeoutError
    Ошибка возникает, если сессия остается бездействующей дольше установленного времени
    ожидания без транзакции
97. IndeterminateCollationError
    Происходит при использовании неопределенной сортировки для сравнения строковых значений
    в запросе
98. IndeterminateDatatypeError
    Ошибка возникает, когда тип данных не может быть определен однозначно в контексте запроса
    или операции
101. InsufficientPrivilegeError
    Происходит, когда у пользователя недостаточно прав для выполнения запрашиваемой операции
103. IntegrityConstraintViolationError
    Общая ошибка для случаев нарушения ограничений целостности (например, уникальность или
    внешний ключ)
104. InterfaceError
    Возникает при проблемах взаимодействия между клиентом и сервером базы данных на уровне
    интерфейса
105. InternalClientError
    Ошибка указывает на внутреннюю проблему клиента базы данных (например, сбой библиотеки)
106. InternalServerError
    Происходит при внутренней ошибке сервера PostgreSQL (например, сбой выполнения запроса)
115. InvalidAuthorizationSpecificationError
    Ошибка возникает, когда спецификация авторизации (например, имя пользователя или пароль)
    недействительна
116. InvalidBinaryRepresentationError
    Происходит, когда строка не может быть преобразована в двоичное представление
119. InvalidCharacterValueForCastError
    Ошибка возникает при попытке преобразования значения символа в другой тип данных, если
    значение не соответствует ожидаемому формату
120. InvalidColumnDefinitionError
    Происходит, когда определение столбца в запросе некорректно (например, неправильный
    тип данных)
121. InvalidColumnReferenceError
    Ошибка возникает при ссылке на столбец, который не существует в таблице или запросе
125. InvalidDatabaseDefinitionError
    Ошибка возникает при некорректном определении базы данных (например, неправильные параметры)
126. InvalidDatetimeFormatError
    Происходит при неверном формате даты и времени в запросе
137. InvalidNameError
    Ошибка возникает при использовании недопустимого имени объекта (например, таблицы или столбца)
138. InvalidObjectDefinitionError
    Происходит, когда определение объекта (например, таблицы или индекса) некорректно
139. InvalidParameterValueError
    Ошибка возникает при передаче недопустимого значения параметра в запросе
140. InvalidPasswordError
    Происходит, если указанный пароль неверен для аутентификации пользователя
144. InvalidRegularExpressionError
    Происходит при наличии некорректного регулярного выражения в запросе
146. InvalidRowCountInLimitClauseError
    Происходит, если значение в операторе LIMIT некорректно
153. InvalidSqlstateReturnedError
    Ошибка возникает, если возвращаемый код состояния SQL некорректен или не соответствует стандарту
154. InvalidTableDefinitionError
    Происходит при некорректном определении таблицы (например, неправильные типы данных или
    ограничения)
157. InvalidTextRepresentationError
    Ошибка возникает при попытке преобразования текста в другой тип данных, если текст не
    соответствует ожидаемому формату
159. InvalidTransactionInitiationError
    Ошибка возникает при попытке начать транзакцию в неподходящем состоянии
162. InvalidUseOfEscapeCharacterError
    Происходит, когда символ экранирования используется неправильно в строковом литерале
178. NoDataFoundError
    Происходит при выполнении запроса, который не возвращает никаких данных (например, поиск
    по несуществующему идентификатору)
184. NotNullViolationError
    Происходит при попытке вставить или обновить запись с NULL-значением в столбце, где это
    запрещено ограничением NOT NULL
187. NullValueNotAllowedError
    Ошибка возникает при попытке вставить значение NULL в столбец или параметр запроса, где
    это запрещено
188. NumericValueOutOfRangeError
    Происходит при выходе числового значения за пределы допустимого диапазона для данного
    типа данных
195. PostgresConnectionError
    Ошибка указывает на проблемы с подключением к серверу PostgreSQL (например, неправильные
    учетные данные)
196. PostgresError
    Общая ошибка для всех типов ошибок PostgreSQL без конкретной классификации
197. PostgresFloatingPointError
    Ошибки связанные с проблемами обработки чисел с плавающей запятой (например, переполнение
    или потеря точности)
199. PostgresSyntaxError
    Возникает при наличии синтаксической ошибки в SQL-запросе
200. PostgresSystemError
    Общая ошибка для системных ошибок PostgreSQL (например, ошибки файловой системы)
201. ProgramLimitExceededError
    Ошибка возникает, когда превышены лимиты программы, установленные в PostgreSQL максимальное
    количество подключений
212. RestrictViolationError
    Ошибка возникает при нарушении ограничений RESTRICT, например, при попытке удалить запись,
    на которую ссылается внешний ключ
219. SQLRoutineError
    Общая ошибка для проблем с вызовом SQL-процедур или функций
231. StringDataLengthMismatchError
    Ошибка возникает при несоответствии длины строки ожидаемой длине (например, если строка
    слишком длинная для столбца)
236. TooManyColumnsError
    Ошибка возникает при попытке создать таблицу с слишком большим количеством столбцов
237. TooManyConnectionsError
    Происходит, когда количество подключений к базе данных превышает установленный лимит
240. TooManyRowsError
    Ошибка возникает при превышении лимита количества строк в результате запроса или операции
249. UndefinedColumnError
    Происходит при обращении к столбцу, который не существует в таблице или запросе
252. UndefinedObjectError
    Происходит при обращении к объекту (например, таблице, индексу или представлению), который
    не существует в базе данных
254. UndefinedTableError
    Происходит при обращении к таблице, которая не существует в базе данных
255. UniqueViolationError
    Ошибка возникает при попытке вставить или обновить запись, которая нарушает ограничение
    уникальности (UNIQUE) на уровне таблицы
256. UnknownPostgresError
    Общая ошибка для случаев, когда возникла неизвестная ошибка PostgreSQL, которая не попадает
    под другие категории
258. UnsupportedClientFeatureError
    Ошибка возникает, когда клиентская программа пытается использовать функцию или возможность,
    не поддерживаемую сервером PostgreSQL
259. UnsupportedServerFeatureError
    Происходит при обращении к функции или возможности сервера, которая не поддерживается
    текущей версией PostgreSQL
265. ZeroLengthCharacterStringError
    Происходит при попытке использования строки нулевой длины там, где это недопустимо
