ASYNCPG:
1. ActiveSQLTransactionError
Ошибка возникает, когда вы пытаетесь выполнить SQL-запрос, но уже существует активная транзакция. Это может произойти, если вы не завершили предыдущую транзакцию перед началом новой.
2. AdminShutdownError
Эта ошибка возникает, когда сервер PostgreSQL был остановлен администратором. Это может произойти, если администратор выполняет команду для завершения работы сервера.
3. AmbiguousAliasError
Ошибка возникает, когда в SQL-запросе используется неоднозначное имя (псевдоним) для столбца или таблицы, что приводит к неясности о том, к какому объекту относится псевдоним.
4. AmbiguousColumnError
Происходит, когда в запросе есть несколько столбцов с одинаковыми именами из разных таблиц, и PostgreSQL не может определить, какой именно столбец вы имеете в виду.
5. AmbiguousFunctionError
Ошибка возникает, если в запросе вызывается функция с одинаковым именем, но различающимися параметрами, и PostgreSQL не может определить, какую именно функцию использовать.
6. AmbiguousParameterError
Происходит, когда параметры функции или запроса имеют одинаковые имена, и PostgreSQL не может определить, какой параметр вы имеете в виду.
7. ArraySubscriptError
Ошибка возникает при попытке доступа к элементу массива по индексу, который выходит за пределы допустимого диапазона.
8. AssertError
Эта ошибка указывает на то, что не выполнено условие утверждения (assert), что может свидетельствовать о логической ошибке в коде.
9. BadCopyFileFormatError
Ошибка возникает при попытке загрузить данные из файла с неправильным форматом для операции COPY.
10. BranchTransactionAlreadyActiveError
Происходит, когда вы пытаетесь начать новую ветвь транзакции (branch transaction), но уже существует активная ветвь.
11. CannotCoerceError
Ошибка возникает, когда PostgreSQL не может преобразовать значение одного типа данных в другой тип данных.
12. CannotConnectNowError
Эта ошибка указывает на то, что соединение с базой данных не может быть установлено в данный момент (например, из-за превышения лимита подключений).
13. CantChangeRuntimeParamError
Ошибка возникает при попытке изменить параметры выполнения сервера PostgreSQL во время его работы.
14. CardinalityViolationError
Происходит при нарушении ограничений кардинальности (например, при попытке вставить больше строк в таблицу, чем это разрешено).
15. CaseNotFoundError
Ошибка возникает при попытке обращения к регистронезависимому имени объекта (например, таблицы или столбца), которое не найдено в базе данных.
16. CharacterNotInRepertoireError
Эта ошибка указывает на то, что символ не поддерживается кодировкой базы данных.
17. CheckViolationError
Происходит при нарушении ограничения проверки (CHECK) на уровне таблицы.
18. ClientCannotConnectError
Ошибка возникает, когда клиентская программа не может установить соединение с сервером базы данных по различным причинам (например, неправильные учетные данные).
19. ClientConfigurationError
Указывает на ошибку конфигурации клиента базы данных, которая мешает успешному подключению или выполнению запросов.
20. CollationMismatchError
Ошибка возникает при несоответствии сортировки между столбцами или выражениями в запросе.
21. ConfigFileError
Происходит при ошибках в конфигурационном файле PostgreSQL.
22. ConfigurationLimitExceededError
Ошибка указывает на то, что превышены лимиты конфигурации PostgreSQL (например, максимальное количество подключений).
23. ConnectionDoesNotExistError
Возникает при попытке взаимодействия с соединением, которое было закрыто или никогда не существовало.
24. ConnectionFailureError
Эта ошибка указывает на общую проблему подключения к серверу базы данных.
25. ConnectionRejectionError
Ошибка возникает, когда сервер отклоняет запрос на подключение из-за настроек безопасности или превышения лимитов.
26. ContainingSQLNotPermittedError
Происходит при попытке выполнить SQL-запрос внутри контекста или блока кода, где это запрещено.
27. CrashShutdownError
Ошибка указывает на то, что сервер PostgreSQL был аварийно завершен и теперь находится в состоянии "краш-выключения".
28. DataCorruptedError
Происходит при обнаружении поврежденных данных в базе данных.
29. DataError
Общая ошибка для случаев некорректных данных (например, неверный формат даты).
30. DatabaseDroppedError
Ошибка возникает при обращении к базе данных, которая была удалена.
31. DatatypeMismatchError
Ошибка возникает, когда тип данных, переданный в запросе, не соответствует ожидаемому типу данных в базе данных.
32. DatetimeFieldOverflowError
Происходит, когда значение даты или времени превышает допустимые пределы для типа данных datetime.
33. DeadlockDetectedError
Эта ошибка возникает, когда две или более транзакции блокируют друг друга, и система не может разрешить ситуацию. PostgreSQL автоматически откатывает одну из транзакций для устранения взаимной блокировки.
34. DependentObjectsStillExistError
Ошибка возникает при попытке удалить объект (например, таблицу или схему), на который ссылаются другие объекты (например, внешние ключи).
35. DependentPrivilegeDescriptorsStillExistError
Происходит при попытке удалить привилегии, на которые ссылаются другие объекты.
36. DiagnosticsError
Ошибка возникает при проблемах с диагностикой состояния базы данных.
37. DiskFullError
Эта ошибка указывает на то, что на диске недостаточно места для выполнения операции.
38. DivisionByZeroError
Происходит при попытке деления на ноль в SQL-запросе.
39. DuplicateAliasError
Ошибка возникает, когда в запросе используются дублирующиеся псевдонимы для столбцов или таблиц.
40. DuplicateColumnError
Происходит при попытке создать таблицу с дублирующимися именами столбцов.
41. DuplicateCursorError
Ошибка возникает при попытке создать курсор с именем, которое уже используется.
42. DuplicateDatabaseError
Происходит при попытке создать базу данных с именем, которое уже существует.
43. DuplicateFileError

44. DuplicateFunctionError

45. DuplicateJsonObjectKeyValueError
Ошибка возникает при наличии дублирующих ключей в JSON-объекте.
46. DuplicateObjectError
Общая ошибка для случаев, когда вы пытаетесь создать объект (таблицу, индекс и т.д.) с именем, которое уже существует.
47. DuplicatePreparedStatementError
Происходит при попытке создать подготовленный оператор с именем, которое уже существует.
48. DuplicateSchemaError
Ошибка возникает при создании схемы с именем, которое уже существует.
49. DuplicateTableError
Происходит при попытке создать таблицу с именем, которое уже существует в базе данных.
50. ErrorInAssignmentError
Ошибка возникает при некорректном присваивании значений в SQL-запросе.
51. EscapeCharacterConflictError

52. EventTriggerProtocolViolatedError
Ошибка возникает при нарушении протокола событийного триггера в PostgreSQL.
53. ExclusionViolationError
Происходит при нарушении ограничения исключения (EXCLUDE) на уровне таблицы.
54. ExternalRoutineError
Ошибка возникает при проблемах с выполнением внешней процедуры или функции.
